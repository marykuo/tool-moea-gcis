# tool-moea-gcis

[![python](https://img.shields.io/badge/python-3.9+-blue.svg)](https://www.python.org/)
[![uv](https://img.shields.io/endpoint?url=https://raw.githubusercontent.com/astral-sh/uv/main/assets/badge/v0.json)](https://github.com/astral-sh/uv)

Python package for fetching and updating GCIS (MOEA) COD data via API.

## Quick Start

This project is managed using [uv](https://github.com/astral-sh/uv). Follow the commands below to get started:

```bash
# Install dependencies and update the lockfile
uv sync

# Execute data update
uv run main

# Linting and automatic formatting
uv run ruff check
uv run ruff format
```
